# Awilli98-web-project
## Project 2 - HTML Foundation Page

For Project 2, I created a basic two-page website using only HTML. My website theme is Gaming As a Mom. I included headings, paragraphs, links, navigation, a list, a table, and an image.

While creating this project, I learned how to organize a basic webpage and link multiple pages together. I also learned how file paths work when pages and images are in different folders.

One challenge I faced was making sure my HTML file names and paths were correct. Once I fixed that, the pages linked together correctly.
